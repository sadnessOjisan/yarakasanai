# yarakasanai

[本番環境でやらかしちゃった人 Advent Calendar 2019](https://qiita.com/advent-calendar/2019/yarakashi-production) 23 日目でやらかしたことを起こさないためのレポジトリです。

```zsh
$ yarn add @sadnessOjisan/yarakasanai
```

## 環境構築

```zsh
yarn install
```

## やらかしたとき

```zsh
npm unpublish yarakasanai --force
```
